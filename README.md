# Purchase Propensity Model

Goal: Given a user's browsing behavior, predict the probability that they purchase from each product category.

The feature window (Oct 1–31) and target window (Nov 1–5) are kept strictly separate. Labels are built on Nov 1–5 purchasers only.

> **Work in progress.** EDA complete, feature engineering and modeling in progress.

## Data

The project uses [the REES46 e-commerce behavior dataset](https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store) which includes two months of user clickstream. Download `2019-Oct.csv` and `2019-Nov.csv` into `data/` to get started.
